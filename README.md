# Career-Sync AI Analyzer 🚀

A Data Analytics–driven tool that compares Resume skills with Job Description
and visualizes skill gaps using a Streamlit dashboard.

## Features
- Resume vs Job skill comparison
- Skill match percentage
- Missing skills recommendation

## Tech Stack
- Python
- Streamlit

## Use Case
Helps students and job seekers identify skill gaps for targeted learning.
